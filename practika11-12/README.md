# Практические занятия 11-12 — Автосалон BMW (RBAC)

Серверное приложение на Node.js с системой ролей (RBAC), JWT аутентификацией и управлением пользователями.

## Функциональность

### Роли пользователей
- **Гость** — неавторизованный пользователь (только регистрация и вход)
- **Пользователь** — просмотр товаров
- **Продавец** — просмотр, создание и редактирование товаров
- **Администратор** — полный доступ (управление товарами + управление пользователями)

### API Маршруты

| Метод | Маршрут | Доступ | Описание |
|-------|---------|--------|----------|
| POST | /api/auth/register | Гость | Регистрация |
| POST | /api/auth/login | Гость | Вход в систему |
| POST | /api/auth/refresh | Гость | Обновление пары токенов |
| GET | /api/auth/me | Пользователь+ | Получение текущего пользователя |
| GET | /api/users | Администратор | Список всех пользователей |
| GET | /api/users/:id | Администратор | Получить пользователя по id |
| PUT | /api/users/:id | Администратор | Обновить информацию пользователя |
| DELETE | /api/users/:id | Администратор | Удалить/заблокировать пользователя |
| POST | /api/users/seller | Администратор | Создать продавца |
| POST | /api/products | Продавец+ | Создать товар |
| GET | /api/products | Пользователь+ | Получить список товаров |
| GET | /api/products/:id | Пользователь+ | Получить товар по id |
| PUT | /api/products/:id | Продавец+ | Обновить параметры товара |
| DELETE | /api/products/:id | Администратор | Удалить товар |
