Прогрессивное веб-приложение для заметок

Данное приложение разработано в рамках курса **«Фронтенд и бэкенд разработка»** (4 семестр, 2025/2026 уч. год). Проект демонстрирует создание современного прогрессивного веб-приложения (PWA) с полным офлайн-функционалом, поддержкой установки на устройство, WebSocket-синхронизацией в реальном времени и push-уведомлениями.

Реализованные практические работы

Практическое занятие №13 — Service Worker
- Регистрация Service Worker для работы в фоновом режиме
- Кэширование статических ресурсов (Cache First стратегия)
- Полная автономная работа приложения без интернета
- Сохранение заметок в localStorage

Практическое занятие №14 — Web App Manifest
- Создание manifest.json с метаданными приложения
- Подготовка иконок всех необходимых размеров (16x16 — 512x512)
- Настройка мета-тегов для iOS (apple-touch-icon, apple-mobile-web-app-capable)
- Возможность установки приложения на домашний экран

Практическое занятие №15 — HTTPS + App Shell
- Настройка локального HTTPS через mkcert
- Реализация архитектуры App Shell (каркас + динамический контент)
- Стратегии кэширования: Cache First для статики, Network First для контента
- Навигация между страницами без перезагрузки

Практическое занятие №16 — WebSocket + Push
- Node.js сервер на Express + Socket.IO
- Двусторонняя синхронизация заметок в реальном времени между вкладками/клиентами
- Генерация и использование VAPID-ключей
- Push-уведомления через Web Push API
- Кнопки включения/отключения уведомлений с сохранением подписки на сервере

Практическое занятие №17 — Детализация Push (напоминания)
- Две формы: простая заметка и заметка с напоминанием
- Выбор даты/времени для напоминания (datetime-local)
- Серверное планирование push-уведомлений через setTimeout
- Хранение активных таймеров в Map на сервере
- Кнопка "Отложить на 5 минут" в push-уведомлении
- Эндпоинт /snooze для переноса напоминания
- Напоминания работают даже при закрытом приложении
